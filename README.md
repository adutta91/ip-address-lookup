
### Step 1 - Installation
```
    $ npm i -g nodemon mocha watchify browserify forever
    $ npm install
```

### Step 2 - Run one of followings
    
```
    // Watch (dev)
    $ npm run watch
```

    
```
    // Build (prod)
    $ npm run build
```

### Step 3 - Start App
```
    $ npm start
```

### Unit Test
```
    $ npm run test
```
